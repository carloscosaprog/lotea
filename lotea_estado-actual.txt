Proyecto LOTEA - Estado actual del desarrollo

Puntos acabados o en funcionamiento:

Backend:
- Sistema de autenticación completo con registro y login (email o nombre de usuario)
- Encriptación de contraseñas con bcrypt
- Generación y validación de token JWT
- Endpoint para obtener y actualizar el perfil del usuario autenticado
- CRUD completo de lotes:
	- Crear lote con subida de imágenes mediante multer
	- Obtener todos los lotes
	- Obtener un lote por ID con sus imágenes
	- Actualizar lote (incluyendo imágenes)
	- Eliminar lote (con control de propietario)
- Relación entre usuarios, lotes, categorías e imágenes
- Endpoint para obtener los lotes de un usuario
- Endpoint para obtener los lotes del usuario autenticado (mis lotes)


Frontend:
- Sistema de autenticación funcional con contexto global (login, logout, persistencia)
- Navegación mediante React Router
- Página Home con listado de lotes en formato grid responsive
- Componente LoteCard reutilizable con imagen, precio y cantidad
- Página de detalle de lote con:
	- Galería de imágenes (miniaturas + fullscreen)
	- Navegación entre imágenes
	- Información completa del lote (precio, cantidad, categoría, descripción)
	- Visualización del vendedor con acceso a su perfil
	- Mostrar más lotes del mismo vendedor
	- Opciones de editar y eliminar si el usuario es propietario
- Página de creación de lote con subida de imágenes
- Página de edición de lote
- Página de perfil de usuario con:
	- Visualización de datos
	- Edición del nombre
	- Cierre de sesión
	- Acceso a "Mis lotes"
- Página de lotes del usuario (mis lotes)
- Navbar funcional con:
	- Acceso a login/registro si no hay sesión
	- Botón "Vender" si el usuario está autenticado
	- Avatar que dirige al perfil
